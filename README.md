# SliceFeatures

This repo takes the full features file and slices it according to features...

---

## Usage

Activate the venv

```shell
source .venv/bin/activate
```

Then run the main script:

```shell
python SLICE.py
```

The script will search in the `INPUT/` folder for two files: the propper `.csv` file and a names file (`.txt`).

The outputted files will be stored in `OUTPUT/`.
